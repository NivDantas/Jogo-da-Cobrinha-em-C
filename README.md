# Jogo-da-Cobrinha-em-C
Criação de um pequeno projeto de Jogo da Cobrinha, utilizando a linguagem C.
## Descrição

Este é um simples jogo da cobrinha implementado em C. O jogador controla a cobra usando as setas do teclado e o objetivo é comer as frutas e aumentar a pontuação sem colidir com as paredes ou com o próprio corpo da cobra.

## Recursos

- Gráficos ASCII para representar a cobra, a fruta e o ambiente do jogo.
- Movimentação suave da cobra em quatro direções: esquerda, direita, cima e baixo.
- Geração aleatória de frutas em posições diferentes a cada vez que uma fruta é comida.
- Aumento do tamanho da cobra ao comer uma fruta.
- Sistema de pontuação para acompanhar o desempenho do jogador.

## Como Executar

1. Certifique-se de ter um compilador C instalado no seu sistema.
2. Clone ou faça o download deste repositório.
3. Navegue até o diretório do jogo da cobrinha em C.
4. Compile o código-fonte do jogo usando o comando: `gcc jogo_da_cobrinha.c -o jogo_da_cobrinha`.
5. Execute o jogo usando o comando: `./jogo_da_cobrinha`.

Ou execute o código em uma IDE com a linguagem C.
